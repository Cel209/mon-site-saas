# Privacy Policy

**Last updated: March 2026**

This Privacy Policy explains how our Discord Bot collects, processes, and protects data. By inviting or interacting with the bot, you consent to the practices described below.

### 1. Data We Collect and Store
The bot operates using local JSON databases and collects the absolute minimum data required for its features:
* **Discord User IDs:** Stored to manage the virtual economy, including Gacha scores, virtual loans, inventory (tickets), active temporary roles, and AI token usage limits.
* **Discord Guild and Channel IDs:** Stored exclusively to configure automated notifications (e.g., free Steam games alerts) and manage support ticket systems.

### 2. Message Content Processing
To provide conversational AI features and image generation, the bot reads message content. 
* **No Persistent Storage:** Message content is **NEVER** logged, saved, or stored in our databases.
* **Third-Party APIs:** Prompts are temporarily transmitted to the Google Gemini API (for text/vision generation) and HuggingFace API (for image generation). Data sent to these APIs is subject to their respective privacy policies.

### 3. Data Protection
Data is stored locally on the host server. We do not sell, trade, or share your Discord IDs or virtual economy profiles with any third-party advertisers or services.

### 4. User Rights and Data Deletion
You have the right to request the complete deletion of your data. If you wish to have your User ID and associated virtual economy data completely purged from our systems, please open a support ticket within the Discord server or contact the developer directly. Data deletion requests are processed within 48 hours.
